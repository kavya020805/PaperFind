# Sprint 3: Frontend Search Interface
**Duration**: Days 6-8

## Goals
Develop a user-friendly React frontend to interact with the search API.

## Tasks
- [ ] Initialize the React application (e.g., using Vite).
- [ ] Build the main Search UI:
  - Implement a search bar component.
  - Design and build result cards displaying title, abstract snippet, and similarity score.
  - Implement keyword highlighting for matched terms.
- [ ] Implement the "Related Papers" feature (fetch nearest neighbors of a given document).
- [ ] Integrate frontend with the Express API (handling auth, search, and pagination).
